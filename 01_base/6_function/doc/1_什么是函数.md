# 1.概述

go函数支持普通函数、匿名函数、闭包。go中函数是“一等公民”，也就说其语句以下特性：

- 1.函数本身可以作为变量
- 2.函数包含了匿名函数、闭包
- 3.函数可以满足接口

# 2.函数的定义
注意：一般而言，函数的参数传递分为值传递和引用传递，<font color="red">但是在go语言中全部都是值传递。</font>
函数定义：
```properties
func 函数名(参数列表)(返回值类型){
    代码
}

说明：
- 参数列表：可选。比如 func add(a int, b int) int { return 0 }, 参数类型相同时可简写为：func add(a,b int) int { return 0 }
- 返回值：可选，支持多个返回值。当无返回 值时，可直接省略“(返回值类型)”；当只有一个返回值时，可省略括号；当有多个返回值时，需保留括号。
  比如func add(a,b int)(int, error) { return a+b, nil }, 也可以指定返回类型的变量名func add(a,b int)(sum int, err error){ return sum, error}
```
# 3.函数的可变参数
使用省略号表示，比如func add(items ... int)(int, error){ return 0,nil }

# 4.匿名函数
定义的函数没有函数名。示例如下：
```go
// 使用变量接收匿名函数  
localFunc := func() {
		fmt.Println("这个是匿名函数")
	}
```
# 5.闭包
定义：一个函数中访问另一个函数的局部变量是不行的，但是在一个函数中定义个匿名函数访问其局部变量是可以的，这就叫闭包。
```go
// 闭包
func autoIncrement() func() int {
	local := 0
	// 定义一个匿名函数
	return func() int {
		local++
		return local
	}
}
```
