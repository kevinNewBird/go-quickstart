# 1.什么是结构体
本质是类型的集合。

# 2.结构体的定义
```properties
type 结构体名 struct {
    类型名 类型
    ... 
}
```
# 3.结构体的使用
## 3.1.结构体的初始化
两种方式：省略属性名和指定属性名的方式。<br/>
- 省略属性名：必须初始化所有的属性值
- 指定属性名：选择性初始化字段
```go
// 1.1.不指定属性名的方式（必须初始化所有字段，否则将会编译报错）
p1 := Person{"tommy", 18, 1.78}
// 1.2.执行属性名的方式（选择性初始化字段）
p2 := Person{name: "tommy"}
fmt.Println(p1, p2)
```
## 3.2.结构的赋值和取值
直接指定属性名的方式进行赋值和取值。

# 4.匿名结构体
临时使用的结构体，定义和使用是同一时间发生的。<br/>
```properties
变量名 := struct{
    定义
}{
    数据
}
```
