# 1.map的概念
map是一个key（索引）和value（值）的无序集合。map定义了必须要初始化，否则其设置将会报错（未初始化其为nil）。<br/>
```go
// 定义但未初始化
var courseMap map[string]string
// 设置值（将会报错）
courseMap["oracle"] = "oracle原理"
```
# 2.map的语法
```properties
var mapVar map[type]type

说明：
- 中括号中的type是key的类型
- 另外一个type则是value的类型
```

# 3.map的初始化
```go

// 1.定义时初始化
var courseMap = map[string]string{}

// 2.使用make内置函数初始化
var courseMap2 = make(map[string]string, 3)
```
注意事项：
- map定义了必须要初始化，否则其设置将会报错（未初始化其为nil）。但是slice是可以只定义并且直接赋值。
# 4.map的遍历
使用range关键字进行遍历。

# 5.map的值获取
支持两种方式：单值返回和两个返回值。

内置函数delete
