# 1.结构体方法的定义
与方法的定义是存在区别的。
```properties
func (s StructType)funcName(param1 param1Type, ...)(returnVal1 returnType1, ...){
   代码逻辑
}

注意：
1.非结构体方法的定义是没有"(s StructType)"的，这一点需要明确 
2.结构体的方式是定义在结构体外面的
3.结构体中的"(s StructType)"也被称为接收器
```
# 2.结构体方法接收器的两种使用方式
- (s StructType): 值传递，副本传递到结构体方法中
- (s *StructType): 指针方式，也叫引用传递
