# 1.结构体嵌套的定义
两种定义方式：嵌套变量和匿名嵌套。
第一种定义方式：取嵌套值比较麻烦，比如stu.p.name<br/>
```go
type Person struct {
	name string
	age int
}

type Student struct {
	// 定义一个属性变量
	p Person
	score float32
}
```
第二种定义方式： 匿名嵌套的方式，可以直接访问嵌套属性，比如stu.name
```go
type Person struct {
	name string
	age int
}

type Student struct {
// 匿名变量
Person
score float32
}
```
