# 1.说明
指针就是地址。一个指针变量指向了一个值的内存地址，类似于变量和常量。使用指针前需要声明指针，声明格式如下：
```go
var var_name *var_type

说明：
var_type: 指针类型
var_name: 指针变量名
*： 用于指定变量作为一个指针
```
# 2.特性
在go语言中，指针存在以下特性：
- a.指针是不能直接参与运算的（c/c++可以执行指针的加减改变指向地址）
- b.指针可以通过unsafe包下的方法实现指针的运算

# 3.初始化
指针必须要初始化，声明方式下指针地址空间为nil，调用其属性值将会报错。<br/>
两种定义方式：
- a.使用变量取地址的方式
- b.使用new(Type)的方式
